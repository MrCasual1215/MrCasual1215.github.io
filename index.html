Projects

 Wheeled Biped Robot: Modeling and Control

HITSZ Auto2003A Innovation Training A

Systematically modeled and linearized the wheel-leg inverted pendulum, applying VMC to derive motor torques and designing an LQR controller. Developed the chassis and electrical system with an STM32 microcontroller, and validated the control system via Simscape Multibody simulations before deployment.

Small-scale Autonomous Driving System

The 18th National Undergraduate Smart Car Contest (Racing Competition)

Developed an autonomous driving system using traffic cones for trajectory tracking, integrating Gmapping for real-time mapping, A* for global path planning, Minimum Snap for trajectory optimization, and Pure Pursuit for local tracking. Employed YOLO for object detection with NPU acceleration to ensure efficient inference.

Motion Target Control and Autonomous Tracking System

2023 National Undergraduate Electronics Design Contest

Designed two laser gimbal systems: one for 2D edge detection using the Sobel operator, and the other for tracking the primary system. Employed a PD controller, achieving precise tracking with an inter-laser error under 0.02 meters.
